# MSPJPWPTHEME
Wordpress custom theme created by Microsoft Student Partners

## REQUIRE PLUGINS

- Advanced Custom Fields
- Crayon Syntax Highlighter
- Custom Post Type Permalinks
- PS Disable Auto Formatting
- WP Masonry Layout
- WP Post Branches
- WP Social Bookmarking Light
- Pz-LinkCard
- TinyMCE Advanced
